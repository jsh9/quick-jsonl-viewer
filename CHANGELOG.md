# Changelog

All notable changes to Quick JSONL Viewer are documented in this file.

## 0.0.7

- Keep JSONL diffs in VS Code's native diff editor while continuing to open JSONL files in Quick JSONL Viewer.

## 0.0.6

- Add an info-bar separator before the rendered preview status text.

## 0.0.5

- Add installation and marketplace information to the README introduction.

## 0.0.4

- Refactor the JSONL viewer implementation and expand coverage across fallback paths.

## 0.0.3

- Focus the JSONL viewer when it opens so VS Code's webview Find shortcut works without first clicking inside the editor.

## 0.0.2

- Enable VS Code's webview Find widget for searching rendered JSONL viewer rows.

## 0.0.1

- Initial release of Quick JSONL Viewer.
- Add a readonly custom editor for `.jsonl` files.
- Add Pretty print, Wrapped raw, and Raw line viewing modes.
- Add configurable preview row limits with full-file indexed rendering support.
- Add file metadata display for size, line count, and last modified time.
