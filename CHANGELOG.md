# Changelog

All notable changes to Quick JSONL Viewer are documented in this file.

## 0.0.2

- Enable VS Code's webview Find widget for searching rendered JSONL viewer rows.

## 0.0.1

- Initial release of Quick JSONL Viewer.
- Add a readonly custom editor for `.jsonl` files.
- Add Pretty print, Wrapped raw, and Raw line viewing modes.
- Add configurable preview row limits with full-file indexed rendering support.
- Add file metadata display for size, line count, and last modified time.
